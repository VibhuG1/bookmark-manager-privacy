Privacy Policy for AI Bookmark Manager Pro

Effective Date: December 24, 2024

Overview

AI Bookmark Manager Pro is committed to protecting your privacy while providing efficient bookmark management capabilities. This privacy policy explains our data practices, focusing on what information we collect, why we collect it, and how it is used to provide our service.

Important Note: All user data is stored EXCLUSIVELY in Chrome's local storage on your device. We do not maintain any external servers, databases, or cloud storage. We do not collect, transmit, or store any user data on our systems. The only data storage occurs within your Chrome browser's built-in storage mechanisms.

Data Collection and Storage

Essential Data Collection

To provide bookmark management functionality, we collect and store the following data locally:

  • Bookmark URLs and titles
  • User-created folder structures and tags
  • Page metadata (descriptions and favicons)
  • User-provided notes and annotations
  • AI-generated page summaries
  • Creation and modification timestamps

Storage Methods

Data Storage Location and Methods:

  • Chrome's Local Storage: Primary storage for bookmark data, which remains exclusively on your device
  • Chrome's Sync Storage: Optional synchronization across your devices (entirely user-controlled and managed through Chrome's built-in sync infrastructure)

Important Technical Details:

  • All data operations (creating, reading, updating, deleting) occur directly in your browser
  • No data is ever transmitted to our servers
  • No external databases are used
  • No user analytics or tracking is implemented
  • No cookies are used for data storage

Data Usage and Processing

Detailed Technical Implementation

For complete transparency, here are the specific data elements we store and process:

1. Bookmark Data Structure

  • URL of the bookmarked page
  • Page title and custom bookmark name (if provided)
  • Folder organization information
  • User-assigned tags for categorization
  • Personal notes and annotations
  • Creation and modification timestamps
  • First 1000 characters of page content (stored locally only)
  • Meta descriptions from the webpage

2. Storage Implementation

  • Chrome's Local Storage:
    • Primary data storage using chrome.storage.local API
    • Structured folder-based organization
    • Automatic data cleanup on bookmark deletion
    • No persistent cookies or cached data
  • Chrome's Sync Storage (Optional):
    • Limited to 8KB per item for cross-device synchronization
    • Only essential bookmark data is synced
    • Sync can be disabled at any time

3. Data Processing Operations

The extension performs these specific operations with your data:

  • Page Content Processing:
    • Extracts page title, meta description, and limited content
    • Generates AI summaries for better organization
    • Processes page content strictly within the browser
  • Search and Organization:
    • Enables full-text search across stored bookmarks
    • Provides folder and tag-based organization
    • All search operations occur locally
  • Data Export:
    • Allows complete data export in JSON format
    • Export includes all user-created content
    • Export function runs entirely client-side

Data Sharing and Third-Party Services

External Service Usage (Limited to Essential Functions Only):

  • AI Service: Used solely for generating page summaries. Only the page content you choose to bookmark is sent to this service, and no personal data or browsing history is shared.
  • Google Chrome APIs: Used exclusively for local storage and synchronization through Chrome's built-in features. We use:
    • chrome.storage.local: For storing bookmarks on your device
    • chrome.storage.sync: For optional Chrome sync features
    • chrome.tabs: For reading current tab information when bookmarking
  • Google Favicon Service: Used only for displaying website icons in your bookmark list. Only the domain names of bookmarked sites are shared with this service.

No other external services or third-party integrations are used. The extension operates entirely within your Chrome browser, using only the above services which are essential for core bookmark functionality.

We do not sell, rent, or share your data with any third parties. Any data transmission to third-party services is limited to what is strictly necessary for the extension's core functionality.

User Rights and Controls

You maintain full control over your data with the ability to:

  • Export your complete bookmark data at any time
  • Delete individual bookmarks or all stored data
  • Enable or disable Chrome sync functionality
  • Manage storage permissions through Chrome's settings

Data Security

We protect your data through:

  • Local storage encryption via Chrome's built-in security
  • Sync data encryption through Chrome's sync infrastructure
  • Minimal data transmission, limited to essential extension functions

Data Retention

Data is retained only as long as necessary for providing bookmark functionality:

  • Local data persists until manually deleted or the extension is uninstalled
  • Sync data remains until removed from Chrome sync or the extension is uninstalled
  • No backup copies are maintained on external servers

Changes to This Policy

We may update this privacy policy to reflect changes in our practices or for other operational, legal, or regulatory reasons. The current version will always be available through the Chrome Web Store and within the extension.

Compliance

AI Bookmark Manager Pro is developed and maintained in compliance with:

  • Chrome Web Store Developer Program Policies
  • Google's User Data Privacy Policy
  • Applicable data protection regulations
